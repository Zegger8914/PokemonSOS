You can insert whatever numbers you want - numbers.
If you input any text, it just doesn't work.


-Pressing the Turn button adds one to the current Turn value.

-Pressing the Chain button adds one to the current Chain value.

-Pressing the PP button subtracts one to the current PP value.

-Pressing the Reset PP button resets the Current PP value to the value of PP Max.

-Pressing the Turn+Chain+PP button adds one to Turn and Chain and subtracts one from current PP.

-Pressting the Turn+PP button adds one to Turn and subtracts one from current PP
.

Lastly, Lock input fields will lock all fields, meaning you can't type on them.  However, the buttons will still work, this is made to avoid pressing on the keyboard and screwing the fields.
